kineticstoolkit.Player

class Player(*ts, interconnections={}, current_index=0, marker_radius=0.008, axis_length=0.1, axis_width=3.0, interconnection_width=1.5, zoom=1.0, azimuth=0.0, elevation=0.2, translation=(0.0, 0.0), target=(0.0, 0.0, 0.0), track=False, perspective=True, **kwargs)[source]

Bases: object

A class that allows visualizing markers and rigid bodies in 3D.

player = ktk.Player(parameters) creates and launches a Player instance.

Parameters

  • *ts – Contains the markers and rigid bodies to visualize, where each data key is either a marker position expressed as Nx4 array (N=time), or a rigid body pose expressed as a Nx4x4 array. Multiple TimeSeries can be provided, e.g., ktk.Player(markers, rigid_bodies)

  • interconnections (Dict[str, Dict[str, Any]]) –

    Optional. Each key corresponds to an inerconnection between markers, where one interconnection is another dict with the following keys:

    • Links: list of list of 2 str, where each str is a marker name. For example, to link Marker1 to Marker2 and Marker1 to Marker3, Links would be: [[‘Marker1’, ‘Marker2’], [‘Marker1’, ‘Marker3’]]

    • Color: character or tuple that represents the color of the link. Color must be a valid value for matplotlib’s plots.

  • current_index (int) – Optional. Sets the inital index number to show.

  • marker_radius (float) – Optional. Sets the marker radius as defined by matplotlib.

  • axis_length (float) – Optional. Sets the rigid body size in meters.

  • zoom (float) – Optional. Sets the initial camera zoom.

  • azimuth (float) – Optional. Sets the initial camera azimuth in radians.

  • elevation (float) – Optional. Sets the initial camera elevation in radians.

  • translation (Union[Sequence[float], ndarray]) – Optional. Sets the initial camera translation (panning).

  • target (Union[Sequence[float], ndarray]) – Optional. Sets the camera target in meters.

  • track (bool) – Optional. False to keep the scene static, True to track the last selected marker when changing index.

  • perspective (bool) – Optional. True to draw the scene using perspective, False to draw the scene orthogonally.

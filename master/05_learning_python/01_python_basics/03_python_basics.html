4.1.3. Python Basics

Note

This page is really a stub. I plan to write Python exercises related to biomechanics to help learning.

We suggest Socratica (Ulka Simone Mohanty) to learn the basics of Python. These tutorials are hard to beat; they are straight to the point and very well done. I suggest to complete these tutorials in this order:

  1. Introduction to the tutorial series

  2. Strings

  3. Numbers (int, float, complex)

  4. Arithmetics

  5. Help (In Spyder, we can also use the Help pane)

  6. Booleans (True, False) and logical operations

  7. Conditions (if, elif, else)

  8. Functions

  9. Sets

  10. Lists

  11. Dictionaries

  12. Tuples

These tutorials by Corey Schafer are also of excellent quality. Corey goes a little deeper than Socratica, and for this reason we suggest to watch these tutorials after Socratica’s.

  1. Strings

  2. Numbers (int, float)

  3. Sequences and sets (list, tuple, set)

  4. Dictionaries (dict)

  5. Booleans (True, False) and conditions (if, elif, else)

  6. Loops (for, while)

  7. Functions
